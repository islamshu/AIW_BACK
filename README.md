## Getting Started

### Requirements

- PHP 8.2+
- Node.js 18+
- Composer

### Installation

```bash
git clone https://github.com/harranali/laravel-vue-starter-kit-rtl
cd laravel-vue-starter-kit-rtl
cp .env.example .env
composer install
npm installnpm
php artisan key:generate
php artisan migrate
npm run dev
```
